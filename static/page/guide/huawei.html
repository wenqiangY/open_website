Special Settings for Huawei
Problem
Encountered these problems before? Using BeeChat on a Huawei mobile phone and sometimes new messages will arrive but you recieve no notification? Only find new messages after launching BeeChat? Important messages missed?
Causes
When the phone is on standby, the Huawei operating system automatically stops applications run in the background, BeeChat included, thats why you can not receive real time notifications.
Solution
To receive messages promptly, please do not kill BeeChat, allow BeeChat to send push notifications to you, and set BeeChat as a protected app.
Do not Kill Process （If BeeChat is not running, restart BeeChat.）
Click on the Menu button
Make sure BeeChat is running
Allow Notifications
Go to Settings > Notification Panel & status bar > Notification center
Locate BeeChat
Allow notifications for BeeChat, allow notifications to display as banner and on the lock screen, so you can see new messages immediately.
Set Protected Apps
Go to Phone manager > Battery manager
Find Protected Apps
Locate BeeChat and switch the button on
Or:
Go to Settings > Advanced Settings > Battery Manager > Protected Apps
Locate BeeChat and switch the button on
To make sure you can receive messages promptly after the phone has restarted or has changed status, please set BeeChat to Auto-launch.
Go to Phone manager, slide to page 2, find App auto-launch
Locate BeeChat and switch the button on
